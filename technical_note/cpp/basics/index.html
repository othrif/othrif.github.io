++*ptr vs. *ptr++

#include <iostream> using namespace std; Rules Prefix operator ++ has the same precedence as dereference: ++*ptr = ++(*ptr). ++ptr evaluates to the incremented value of ptr Postfix operator ++ has a higher precedence than dereference: So *ptr++ gets evaluated as *(ptr++).
