    Substring key match in a dictionary

    12 Apr 2020

    Using items() + list comprehension

    test_dict = {'All' : 1, 'have' : 2, 'good' : 3, 'food' : 4} 
search_key = 'ood'
res = [val for key, val in test_dict.items() if search_key in key] 
res

    [3, 4]
